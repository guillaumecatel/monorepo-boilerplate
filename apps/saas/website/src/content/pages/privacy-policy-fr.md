# Politique de confidentialité

## 1. Informations que nous collectons

Nous collectons les informations suivantes :

- Informations d'identification personnelle (nom, adresse e-mail)
- Données d'utilisation du site web
- Cookies et technologies similaires

## 2. Comment nous utilisons vos informations

Nous utilisons vos informations pour :

- Fournir et maintenir notre service
- Améliorer l'expérience utilisateur
- Communiquer avec vous

## 3. Protection des données

Nous mettons en place des mesures de sécurité appropriées pour protéger vos données personnelles contre l'accès non autorisé, la modification, la divulgation ou la destruction.

## 4. Vos droits

Vous avez le droit de :

- Accéder à vos données personnelles
- Corriger vos données
- Supprimer vos données
- Vous opposer au traitement

## 5. Contact

Pour toute question concernant cette politique de confidentialité, contactez-nous à : privacy@example.com
